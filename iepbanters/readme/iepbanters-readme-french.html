Introduction


Le but de ce mod est de s'assurer que chaque PNJ de Bioware ait une part équitable de bavardages entre eux. Nous avons aussi l'intention d'étendre les amitiés et les relations entre les différents PNJs ; l'amitié entre Mazzy et Aerie, par exemple, ou la relation particulière entre Jaheira et Viconia. Pour le moment, le mod comprend des bavardages principalement pour Shadows of Amn, et deux pour ToB (entre Minsc et Aerie).

Nous espérons que vous apprécierez ce mod autant que nous avons pris plaisir à le créer !

➽ Visitez le site web ou le forum pour obtenir les informations les plus récentes.

Compatibilité


Jeux concernés

IEP Extended Banter est conçu pour fonctionner sur tous les jeux Infinity Engine, ce qui inclut :

  • Baldur's Gate II: Enhanced Edition (BG2EE).
  • Le jeu original Baldur's Gate II (Les Ombres d'Amn : BG2-SoA), avec son extension Trône de Bhaal (Throne of Bhaal : ToB). Assurez-vous que votre jeu utilise bien la dernière version du patch, c'est à dire le patch 26498. Consultez les fichiers ReadMe relatifs à votre patch dans votre répertoire de jeu et le site de BioWare si nécessaire.
  • Les mods de conversion basés sur ToB, Baldur's Gate Trilogy (BGT) (depuis la version 1.1), et Enhanced Edition Trilogy (EET).


Compatibilité avec les autres mods

IEP Extended Banter est un mod WeiDU, il devrait par conséquent être compatible avec tous les mods WeiDU.

Il a été testé sur plusieurs installations et a fonctionné parfaitement jusqu'à présent. Si vous rencontrez des bugs, veuillez les signaler dans le forum, s'il vous plaît.

Ayez à l'esprit que l'accélérateur de discussions d'IEP va désinstaller tout accélérateur de discussions installé au préalable, y compris l'accélérateur inclus dans BG1 NPC. C'est dû à des raisons de compatibilité, mais cela signifie que vous devez faire attention avec des installations importantes comprenant d'autres accélérateurs.


Ordre d'installation

➽ Bien qu'il ne soit pas requis pour faire correctement tourner EP Extended Banter, je vous recommande fortement d'installer la dernière version du BG2 Fixpack avant d'installer ce mod.

  • Veuillez installer IEP Extended Banter après TobEx, BG2 Fixpack et EET.
  • Please install IEP Extended Banter avant EET_End et toutes les routines de "biffing".

Installation


Mise en garde

Si une ancienne version de ce mod est déjà installée, il est nécessaire de la désinstaller d'abord. Pour cela, lancez  setup-iepbanters.exe  (ou .command si vous êtes sous Mac OS) et désinstallez tous les composants précédemment installés. Une fois la désinstallation achevée, supprimez le répertoire 📁 iepbanters.

Lorsque vous installez ou désinstallez, ne fermez pas la fenêtre DOS en cliquant sur le bouton X ! Au lieu de cela, appuyez sur la touche Entrée lorsque l'invite de commandes vous le demande.

Par précaution, désactivez les antivirus ou tout logiciel résidant en mémoire avant d'installer ce mod, ou tout autre mod. Certains (en particulier avast et Norton !) ont une fâcheuse tendance à déclarer les exécutables des mods comme des faux positifs, provoquant ainsi l'échec de la procédure d'installation.

Vous pouvez extraire les fichiers de l'archive à l'aide de 7zip, WinRAR, ZipGenius, ou de toute autre utilitaire de compression de fichiers reconnaissant les fichiers .zip.



Note pour les jeux en Édition Améliorée (EE)

Les Éditions améliorées sont des jeux que le développeur fait encore évoluer, notamment par l'ajout de capacités supplémentaires destinées à la création de mods et par l'ajout de contenus. N'oubliez pas que chaque patch de mise à jour effacera les mods que vous avez installés ! Ce mod ne fera pas exception à la règle.

Si vous pouvez retarder la mise à jour du patch en plein milieu d'un partie moddée (si vous en avez la possibilité, notamment chez Beamdog et Good Old Games), n'oubliez pas que même après avoir réinstallé les mods sur un nouveau patch, vous ne pourrez peut-être pas continuer le jeu avec vos anciennes sauvegardes, en particulier à cause de noms de personnages, de lieux, etc, qui pourraient être incorrects. Pour y remédier, copiez tout le dossier du jeu dans un nouveau dossier dans lequel vous installerez vos mods, et qui ne sera pas modifié par les patches de mise à jour. Il est important que vous installiez le mod dans la version linguistique dans laquelle vous jouez. Sinon, les dialogues du mod ne s'afficheront pas et provoqueront des messages d'erreur.



Windows

IEP Extended Banter pour Windows est livré et installé avec WeiDU, et est diffusé sous forme d'archive compressée.

IEP Extended Banter est un mod WeiDU et est très simple à installer. Vous devez extraire les fichiers de l'archive dans votre répertoire de jeu (le dossier qui contient le fichier CHITIN.KEY). Une fois l'archive extraite correctement, vous devriez trouver le répertoire 📁 iepbanters et le fichier setup-iepbanters.exe file dans votre répertoire de jeu. Pour installer le mod, il suffit de double-cliquer sur  setup-iepbanters.exe  et de suivre les instructions affichées à l'écran.

Vous pouvez lancer  setup-iepbanters.exe  dans votre répertoire de jeu pour réinstaller, désinstaller, ou encore changer des composants.



Mac OS

IEP Extended Banter pour Mac OS est diffusé dans la même archive compressée et installé avec WeiDU.

Extrayez le contenu de l'archive dans votre répertoire de jeu. Après une extraction réussie, ce dernier contiendra les fichiers setup-iepbanters, setup-iepbanters.command, et le répertoire 📁 iepbanters. Pour installer, il suffit de double-cliquer sur  setup-iepbanters.command  et de suivre les instructions affichées à l'écran.

Vous pouvez lancer  setup-iepbanters.command  dans votre répertoire de jeu pour réinstaller, désinstaller ou changer des composants.



Linux

IEP Extended Banter pour Linux est diffusé dans la même archive compressée, mais ne contient pas l'installateur WeiDU.

Extrayez le contenu du mod dans le répertoire du jeu que vous voulez moder.

Téléchargez la version la plus récente de WeiDU pour Linux sur WeiDU et copiez weidU et weinstall dans /usr/bin. Ouvrez ensuite un terminal et allez ( cd ) dans le répertoire d'installation de votre jeu.

Optionnel : lancez « tolower » et répondez « Y » aux deux questions. Vous pouvez éviter d'exécuter la seconde option (Linux.ini) si vous l'avez déjà lancée une fois dans le même dossier. Pour gagner du temps, l'archive est déjà passée en minuscules, aussi il n'est pas non plus nécessaire d'exécuter la première option (passer les noms de fichiers en minuscules) si vous n'avez extrait que ce mod depuis la dernière fois que vous avez passé les noms de fichiers en minuscules. Si vous avez un doute, il vaut mieux exécuter tolower et accepter les deux options.

Lancez  weinstall setup-iepbanters  depuis le dossier de votre jeu pour installer le mod. Puis, lancez  wine bgmain.exe  (ou  wine baldur.exe  pour les jeux EE), et commencez à jouer.



Note pour effectuer une désinstallation complète

En plus des méthodes détaillées plus haut pour supprimer des composants, il est possible de désinstaller complètement le mod en tapant  setup-iepbanters --uninstall  dans une ligne de commandes, ce qui supprimera tous les composants sans devoir ingurgiter tous les messages.

1. Interaction Minsc/Aerie plus développée dans ToB

➽ Ce composant comprend deux bavardages entre Aerie et Minsc dans Throne of Bhaal.



2. Dialogue "rétrospectif" pour Imoen dans ToB

➽ Ce composant ajoute un nouveau dialogue avec votre sœur adoptive Imoen dans Throne of Bhaal.



3. Dialogue au réveil pour la romance d'Anomen (de David Gaider avec des ajouts de Kulyok)

➽ Ce composant ajoute un nouveau dialogue à la romance d'Anomen. Dans la romance d'origine, il n'y avait pas de dialogue du "matin d'après" avec Anomen après avoir couché avec lui, une chose qu'à la fois les fans et l'auteur original de la romance, David Gaider, ne trouvaient pas satisfaisante. Ce composant ajoute un dialogue du "matin d'après" écrit par David Gaider dans les forums Attic, avec des ajouts de Kulyok.



4. Révision des conflits entre PNJs

➽ Ce composant comprend des reprises des conflits qui permettent au personnage principal d'intervenir avant que les désaccords entre PNJs ne tournent au vinaigre. Il révise les conflits suivants :

  • Edwin/Keldorn
  • Edwin/Minsc
  • Edwin/Valygar
  • Keldorn/Viconia


5. Restitution de commentaires Aerie/Jaheira/Anomen dans l'Ombreterre

➽ Ce composant, de façon surprenante, restitue un échange en Ombreterre si vous êtes en romance avec l'une d'elles.
